JavaScript

JavaScript es un lenguaje de programación utilizado para desarrollar sitios web interactivos.

Es ampliamente usado en el desarrollo de aplicaciones web modernas.

Este lenguaje se ejecuta en el navegador y permite la interacción con los usuarios.

Las funciones son uno de los pilares de JavaScript.

Los eventos son esenciales para crear aplicaciones dinámicas e interactivas.

function saludo() { console.log("Hola mundo"); } const x = 5;

CSS

CSS se usa para estilizar las páginas web, controlando su apariencia.

Las hojas de estilo definen cómo se deben mostrar los elementos HTML.

Con CSS, puedes cambiar colores, fuentes y diseño en la página.

body { background-color: #f0f0f0; } p { font-size: 16px; }
